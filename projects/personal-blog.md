## 개요

개발 학습 기록과 기술 글을 정리하기 위한 개인 블로그입니다.

## 주요 기능

- Velite 기반 Markdown 콘텐츠 관리
- 카테고리, 태그, 시리즈 분류
- TIL 달력 뷰
- 다크 모드
- 전문 검색 (⌘K)
- Giscus 댓글
- SEO 최적화
